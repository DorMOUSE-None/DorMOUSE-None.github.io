ASM 核心包基本内容漫谈

2018-06-25 :: fangfeng
#Java  #ASM  #Visitor Pattern 

本文描述的 ASM 指的是 OW2 ASM

ASM-Core 的结构

首先是一些概述性的内容。

由于 ASM 操作的 JAVA 字节码有严格的格式规定，且即使随着 JVM 标准的升级也极少出现重大调整。 因此适用面狭窄的访问者模式在该项目中被大量地使用，并且已经到了丧心病狂的程度:)

从核心包声明的类来看，主要包括:

  1. ClassReader - 作为结构化对象，将接收(accept)访问者的访问

  2. 几种访问者抽象类以及相应的实现类

  3. AnnotationVisitor -> AnnotationWriter

  4. ClassVisitor -> ClassWriter

  5. FieldVisitor -> FieldWriter

  6. MethodVisitor -> MethodWriter

  7. ModuleVisitor -> ModuleWriter

  8. Opcodes & Constants - ClassFile 中描述的大量常量符号与值

  9. 其它一些辅助的类

  10. Attribute - 用于处理非标准化的属性(ClassFile 允许JVMS 中未定义的 Attribute)

  11. ByteArray - 动态可自适应的 byte[] (字节数组)

  12. Context - ClassReader 在被解析(被访问)过程中用于表示“累积状态”的一个类/对象

  13. Symbol - 用于表示 ClassFile 中描述的 Constant 的基类

  14. SymbolTable - 用于存储常量池对象

  15. 其它内容省略
