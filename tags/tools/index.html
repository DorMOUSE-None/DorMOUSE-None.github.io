如何方便地获取 CGlib 生成类

2018-07-13 :: fangfeng
#CGlib  #tools 

配置参数

命令行使用

在 java 启动命令中添加参数配置项 -Dcglib.debugLocation=<Custom Path>

编码实现

在执行 CGlib 获取新生成类之前，调用 System.setProperty("cglib.debugLocation", <Custom Path>)
