理解 Linux Kernel (7) - 字符设备

2018-12-28 :: fangfeng
#Linux  #Kernel  #Char Dev 

相比较于块设备，字符设备无论从物理认知上，抑或是理论理解上，都存在着相当大的入门门槛。特别是在将字符设备与控制台、命令行终端混淆的时候，就更加难以进行分辨了。

回到字符设备本身，字符设备与块设备最主要的区别就在于块设备可以随机读写，而字符设备只能够顺序读，顺序写。

那么，常见的字符设备有什么？显示器、键盘、鼠标。
